# «Игры на коленке» / VIBE JAM — анонс-ролик (производство)

Рилс-анонс рубрики мини-игр. Маскот-сепия выходит на поле → прикидывается мячом → её гоняют → выбивают → дома делает игру → показ игры «Кому продаём?» + CTA. План: [`../../docs/games-on-knee/anons-reel-plan.md`](../../docs/games-on-knee/anons-reel-plan.md).

Две языковые версии = два ролика: **RU** (готова озвучка) и **EN** (в работе). Один видеоряд, разные VO + плашки/тайтлы.

## Пайплайн и инструменты (ВАЖНО)

| Задача | Инструмент | Команда |
|---|---|---|
| Картинки/кадры (Nano Banana) | Gemini 3 Pro Image | `npm run gen:image -- --refs <csv|dir> --aspect 9:16 --model gemini-3-pro-image --out … --prompt "…"` |
| Видео из кадра(ов) | **fal.ai · Seedance 2.0** | `npm run gen:video -- --image start.jpg [--end-image end.jpg] --dur 5 --aspect 9:16 --out …` |
| Озвучка (TTS) | **ElevenLabs** | `npm run gen:voice -- --voice <id> --text "…" --out …` |
| Текстовые плашки/титры | **satori + resvg** | `npx tsx scripts/gen-endcard.ts [--en] [--out …]` |
| Финальная склейка | Kdenlive | вручную |

### Жёсткие уроки (НЕ повторять ошибки)
- **Видео генерим через fal.ai, НЕ через Symphony Creative Studio.** Symphony: (1) игнорирует start/last-frame (морф/переход не выполняет), (2) режет по модерации — **ошибка `10001301` = в кадре дети** (несовершеннолетние). fal.ai режим start+end (`--end-image`) выполняет переходы честно и без модерации.
- **Футболисты — только ВЗРОСЛЫЕ.** Из-за модерации детей все кадры с игроками перегенерены во взрослых (правка через gen:image с рефом-самого-кадра, формулировка «ZERO children, every player adult man»; иногда нужно 2–3 прохода — модель упрямо оставляет ребёнка).
- **Текст НЕ генерим видео/картинко-моделью** (скриншоты, плашки) — будет каша. Скриншоты оживляем детерминированно (ffmpeg Ken Burns/скролл), плашки рендерим satori+resvg.
- `gen:video` ведёт леджер расходов: `assets/video-out/ledger.jsonl` (`npm run gen:video -- --summary`).
- fal-клипы — **720×1280**; демо-игра и плашки — **1080×1920**. При сборке привести к одному (апскейл 720→1080 или общий 1080).

- **Голос сепии (RU): ElevenLabs voice ID `rSfuQoQ3FY8SVKeraMAp`**, модель **`eleven_v3`**.
- Интонация: аудио-теги `[excited] [tired] [sighs] [groggy] [cheerful] [content]` + пунктуация (`…`, `—`) + `--stability` (ниже = эмоциональнее) / `--style`.

- стадион/гул: `…football-game-in-a-big-arena…`, `…stadium-roar…`, `…football-game-sound-effects…`
