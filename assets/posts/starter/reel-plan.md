# Витринный рилс «Стартер» — план съёмки

Сопроводительный файл к [text.md](text.md). Видео со мной на однородном фоне (green-screen workflow → [scripts/chroma-key.ts](../../../scripts/chroma-key.ts)), фон добавляется на монтаже.

---

## Общие параметры

- **Формат:** 9:16, вертикаль.
- **Хронометраж:** цель **60–75 сек** (текущая начитка ~1:30 — на верхней границе; см. ниже как ужать). Витринный/закреплённый рилс досматривают лучше случайного из ленты, но удержание (avg watch time) — главный сигнал алгоритма, поэтому режем «эссеистику».
- **Звук:** обязательно **субтитры** (большинство смотрит без звука).
- **Хук:** первые 2–3 сек — самое цепкое, НЕ раскачка «О, этот дивный мир». Начинаем с темы свободы/«создай своё приложение».
- **Основной план:** средний/погрудный (голова + плечи) — контакт глаза-в-глаза держит доверие в personal-brand и проще кеится. Врезки во весь рост / демо приложения — для динамики.

## Композиция (план)

**Рекомендация: средний/погрудный план как основа + врезки.**

- **Голова + плечи** — главный план: контакт глаза-в-глаза, доверие, проще чистый ключ (волосы по краю — единственная сложность, решается контровиком/боковой подсветкой на фон).
- **Во весь рост** — врезками на блоке про свободу/движение: энергичнее, но лицо мелкое, нужен ровный свет по всей фигуре и больше пространства.
- **Демо-врезки** — экран приложения вместо отдельной карусели (исходная идея реюзится здесь).

## Фон

Метафора в тексте двойная — раскладываем по сегментам:

- **Горы / тропы** → буквально поддерживают финал-метафору: «проводник», «тайные тропы», «короткие пути», «препятствия», «путешествие». Ставим на контекст (2) и оффер (5).
- **Море + пальмы** → бьёт в середину: «свобода выбора места для жизни», удалённый доход. Ставим на блок свободы (3).

Технически:
- Фон **спокойный и слегка размытый** (имитация глубины резкости) — иначе перетягивает внимание с лица.
- Лёгкое **медленное движение** (дрейф облаков / волн), не статика — статичная подложка читается как дешёвый монтаж.
- Если делать один фон на весь рилс — берём **горы** (совпадает с ключевой метафорой проводника).

## Свет / съёмка (для чистого ключа)

- Однородный фон без складок и теней; модель на 1.5–2 м от фона (убрать падающую тень).
- Ровный мягкий свет на лицо (ключ + заполнение), **контровик/раздельная подсветка фона** — отделяет силуэт, спасает кей по краю волос.
- Избегать одежды в цвет фона и глянцевых/прозрачных краёв.
