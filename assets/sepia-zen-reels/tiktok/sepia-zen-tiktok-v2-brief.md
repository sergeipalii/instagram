# Sepia Zen — TikTok ролик v2: бриф для сборки ассетов

Документ для Claude Code. Задача: подготовить ассеты (скриншоты + озвучка + музыка) для
вертикального ролика TikTok про приложение **Sepia Zen** (студия Sepia Software).
Финальный монтаж собирается вручную в Kdenlive (профиль **Vertical HD 30 fps, 1080×1920**).

---

## 0. Контекст и итог v1

Первая версия ролика уже собрана в Kdenlive и состоит из:
- записи экрана приложения (XRecorder, 576×1280, с чёрными полями, слабая динамика) — **заменяем**;
- 3 анимированных фонов от TikTok Symphony (Seedance / Dreamina, по 5 сек каждый);
- титра «Sepia Zen» (огненный цвет) + «App Store · Google Play» в финале;
- озвучки + музыки из записи приложения (первые ~9 сек), музыки на остальное.

v2 повышает качество: убираем слабую запись, строим ролик из качественных скриншотов 1080×1920,
которые «оживают» в анимированные фоны через короткие Seedance-переходы, с единой озвучкой и музыкой.

## 3. Seedance-переходы «оживления» (ключевая новая часть)

Идея: статичный скриншот → он плавно «оживает» → переходит в анимированный фон.

Для каждой пары (скриншот N → анимация N) нужен короткий **5-сек переходный клип**, где
картинка из неподвижной превращается в движущуюся (начальный кадр клипа - статический фон, конечный кадр - первый кадр анимированного фона).

**Делаем платно через Seedance API** (fal.ai)

Файлы переходов: `1-trans.mp4` , `2-trans.mp4` , `3-trans.mp4` (по 5 сек).

## 4. Озвучка (ElevenLabs — у Claude Code есть доступ)

Использовать **тот же voice ID**, что и в приложении Sepia Zen (для аудио-единства с продуктом).
Это легитимно: не клонирование чужого голоса, а генерация нового текста тем же голосом в том же сервисе.

**Параметры генерации:**
- голос: тот же, что в приложении;
- темп: медленный, с паузами (медитативная подача, не тараторить);
- паузы между фразами: разбивать на отдельные строки / точки, при поддержке — добавить `<break time="0.8s"/>`;
- формат вывода: mp3, моно или стерео, 44.1 kHz.

**Текст озвучки (вариант A — нейтральный):**
```
Take a breath.
Let the noise fade.
A moment of calm — anytime you need it.
Sepia Zen.
```

**Текст озвучки (вариант B — в тон аффирмаций):**
```
Slow down.
You are exactly where you need to be.
Find your calm, one affirmation at a time.
Sepia Zen.
```

- Сгенерировать оба варианта, выбрать при монтаже.
- Имена: `voiceover-A.mp3`, `voiceover-B.mp3`.
- Хронометраж держать в пределах ~18–21 сек (под длину ролика). При необходимости сократить текст.

> Без медицинских заявлений («лечит тревогу», «cure anxiety» и т.п.) — TikTok модерирует такое.
> Допустимо: calm, relaxation, mindfulness, a moment of quiet, stress relief.

## 5. Музыка

Использовать **ту же фоновую музыку, что в приложении** — для полного аудио-единства ролика и продукта.

**Проверить лицензию:** убедиться, что лицензия трека покрывает использование в маркетинге / соцсетях,
а не только внутри приложения. Если лицензия ограничена приложением — взять отдельный royalty-free трек
(Pixabay Music / YouTube Audio Library), ambient / meditation.

- Длина: ≥ 22 сек (с запасом под fade out в конце).
- Имя: `music-bg.mp3`.

## 6. Целевая структура ролика (для монтажа в Kdenlive)

Ориентир ~21 сек. Дорожки: V1 — видео, V2 — титр поверх, A1 — озвучка, A2 — музыка.

```
[Сцена 0]  Рука с телефоном → наезд на экран (опц., Seedance image-to-video)   ~3 сек
[Сцена 1]  screen-01 (статика, лёгкий зум)  →  trans-01 (оживление)  →  anim-01   фон
            ── fade/dissolve ──
[Сцена 2]  screen-02  →  trans-02  →  anim-02
            ── fade/dissolve ──
[Сцена 3]  screen-03  →  trans-03  →  anim-03
[Финал]    титр «Sepia Zen» (огненный) + «App Store · Google Play» поверх anim-03/углей
```

- Под всем: `voiceover` (A1) + `music-bg` (A2, приглушённо: −12…−15 dB, тише под голосом).
- Переходы между сценами: dissolve/fade 0.5–0.7 сек (скрывает ИИ-стыки).
- Каждый статичный скриншот: 2–3 сек неподвижно, затем переход «оживления».
